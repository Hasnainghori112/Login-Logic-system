# 🔒 Simple Login System

A basic **Python-based** login authentication system with a limited number of attempts.  

## 🚀 Features
- **Username & Password authentication**
- **Limited login attempts (3)**
- **Error handling for incorrect credentials**
- **Secure exit after max attempts**  

## 🏗️ Technologies Used
- **Python** 🐍  

## 📌 How It Works
1. **User enters a username and password.**  
2. **If correct:** Login is successful.  
3. **If incorrect:** The system reduces attempts and prompts again.  
4. **After 3 failed attempts:** The system locks out the user.
